Page View Track when users view any page 
dataLayer.push({
  'event': 'page_view',
  'page_title': document.title,
  'page_location': window.location.href,
  'page_path': window.location.pathname
});
 page_view
  • page_title
  • page_location
  • page_path
  1. Reports → Engagement
  2. Pages and screens
  3. Or: Life cycle → Acquisition → Traffic acquisition

View Item List Track when users view a list of products 
dataLayer.push({
  'event': 'view_item_list',
  'ecommerce': {
    'item_list_id': 'related_products',
    'item_list_name': 'Related Products',
    'items': [{
      'item_id': 'SKU_12345',
      'item_name': 'Product Name',
      'price': 19.99,
      'item_category': 'Category',
      'index': 1
    }]
  }
});
 view_item_list
  • item_list_id
  • item_list_name
  • items[]
  1. Reports → Engagement → Monetization
  2. Ecommerce purchases
  3. Click "Shopping behavior" template
  4. View "List views" metric
 Fire when product list loads

Creating Custom Reports

To create custom reports for any of these events:

  1. Navigate to Explore → Create new exploration
  2. Select your desired visualization (table, line chart, etc.)
  3. In dimensions, add relevant event parameters
  4. In metrics, add event count or custom metrics
  5. Apply any necessary filters or segments

Implementation Notes

Common Parameters for Items Array
  • item_id (required)
  • item_name (required)
  • price (required)
  • quantity (required for cart/checkout/purchase)
  • item_brand (recommended)
  • item_category (recommended)
Best Practices
  • Clear ecommerce object before pushing new data
  • Use consistent currency codes
  • Include all recommended parameters
  • Test in GA4 DebugView
  • Validate in GTM Preview mode
